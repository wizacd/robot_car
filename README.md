# robot_car
A robotic car powered by an ESP32, controlled via a smartphone using the Dabble GamePad module, with steering provided by a servo motor.

## Source Code
The main source code for this project is available in `robot_car.ino`.

## Features
- Smartphone-based control using Dabble GamePad
- Forward, reverse, and stop motion
- Servo-based steering system
- ESP32 Bluetooth communication
